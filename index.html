you have a search bar for every page so here are the pointer on how to search a particular genre of Manhwa with it:

  • [VR] :- Virtual Reality
  • [COM] :- Comedy
  • [OP] :- Mc OverPowered from the start

You should type these in the mentioned order.
For example, you want to search for a VR Manhwa with OP mc then you have to go like this:
[VR][OP]
Otherwise you won't get the result you want. Like if you search
[OP][VR]
then it will show "Not Found".
